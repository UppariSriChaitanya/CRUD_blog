# Blog Management System

## 📌 Project Overview
A secure Blog Management System developed using PHP and MySQL. Users can register, log in, and perform CRUD operations on blog posts with authentication and security enhancements.

## 🚀 Features
- User Registration
- User Login & Logout
- Dashboard
- Create Blog Post
- Edit Blog Post
- Delete Blog Post
- Search Blog Posts
- Pagination

## 🔒 Security Features
- Prepared Statements (SQL Injection Prevention)
- Password Hashing (`password_hash()`)
- Password Verification (`password_verify()`)
- XSS Protection (`htmlspecialchars()`)
- Session-Based Authentication

## 🛠 Technologies Used
- PHP
- MySQL
- HTML5
- CSS3
- Bootstrap 5
- XAMPP

## 📂 Project Structure
- register.php

- create.php

- index.php
- db.php
- auth.css
- style.css

## ▶️ How to Run
1. Start Apache and MySQL in XAMPP.
2. Import the `blog` database into phpMyAdmin.
3. Place the project folder inside `htdocs`.
4. Open `http://localhost/your_project_folder/` in your browser.
